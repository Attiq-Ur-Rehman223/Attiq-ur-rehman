## Project Overview

LuxeWear is a premium fashion brand that migrated from an aging Magento 2 store to a fully custom Shopify Plus experience. The brief called for a luxury feel that matched their high-end product positioning, combined with modern conversion-focused features.

## Key Features

- **Custom Liquid theme** built from scratch with pixel-perfect Figma implementation
- **3D product viewer** using Three.js for immersive product exploration
- **AR try-on feature** via Shopify's native AR capabilities + custom WebXR layer
- **Klaviyo email flows** including welcome series, abandonment, and post-purchase
- **Custom size guide** with body measurement calculator
- **Advanced filtering** with instant AJAX search powered by Shopify Predictive Search API

## Results

- +340% conversion rate compared to Magento store at launch
- 2.1s average load time (down from 6.8s on Magento)
- 98 PageSpeed score on mobile
- 45% reduction in cart abandonment rate
