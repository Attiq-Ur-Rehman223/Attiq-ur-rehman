# Attiq Ur Rehman — Shopify & WordPress Developer Portfolio

A modern, dark-themed portfolio website for a Shopify and WordPress developer / eCommerce specialist. Built with TanStack Start and deployed on Netlify.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | TanStack Start (SSR) |
| Frontend | React 19, TanStack Router v1 |
| Build | Vite 7 |
| Styling | Tailwind CSS 4 |
| Content | Content Collections (type-safe markdown) |
| Forms | Netlify Forms |
| Deployment | Netlify |
| Language | TypeScript 5.7 (strict mode) |

## Pages

| Route | Description |
|-------|-------------|
| `/` | Homepage — hero, stats, services overview, featured projects, testimonials, CTA |
| `/about` | About page — bio, skills, work experience, certifications |
| `/services` | Services page — 9 service offerings with features, pricing, and process |
| `/projects` | Portfolio — filterable project showcase with results highlights |
| `/resume` | Resume page — work history and education from content collections |
| `/contact` | Contact form with Netlify Forms integration |

## Running Locally

```bash
npm install
npm run dev
```

The dev server runs on `http://localhost:3000` (or port 8888 via Netlify CLI for full platform emulation including forms).

```bash
# With Netlify CLI (recommended for forms testing)
netlify dev
```

## Content Management

All content is managed via Markdown files in the `content/` directory:

- `content/jobs/` — Work experience entries
- `content/education/` — Education history
- `content/projects/` — Portfolio project case studies
- `content/blog/` — Blog posts

Content schemas are defined in `content-collections.ts` using Zod.

## Environment Variables

No environment variables are required for the base portfolio. If you extend the site with the AI assistant feature, add:

```
ANTHROPIC_API_KEY=your_key_here
```

## Deployment

The site deploys automatically to Netlify. Configuration is in `netlify.toml`.

- Build command: `vite build`
- Publish directory: `dist/client`
- SSR function: auto-generated by TanStack Start's Netlify plugin
